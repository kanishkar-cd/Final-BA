ere are the step-by-step commands to set up, install, and run the project inside VS Code on Windows.

1. Open the Project in VS Code
Open VS Code, press Ctrl + ` to open the terminal, and navigate to the project repository:

powershell


cd c:\Users\ASNaviyaDharshini\Desktop\user_story\cd-se-accelerators
2. Activate the Virtual Environment
Activate the existing virtual environment v located in the parent directory:

powershell


..\v\Scripts\Activate.ps1
3. Install/Restore Dependencies
If you need to ensure all required python packages are installed, navigate to the backend folder and run pip install:

powershell


cd backend
pip install -r requirements.txt
4. Configure Environment Variables
Set the required environment variables in your PowerShell session:

powershell


$env:GROQ_API_KEY="YOUR_GROQ_API_KEY"
$env:MODEL_PROVIDER="groq"
$env:MODEL_NAME="llama-3.3-70b-versatile"
$env:QDRANT_HOST=":memory:"
NOTE

Setting QDRANT_HOST=":memory:" allows you to run the RAG search and indexing features in standalone in-memory mode without needing any external Qdrant Docker containers running.

5. Start the FastAPI Server
Run the FastAPI application locally using Uvicorn:

powershell


uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
6. Verify and Test
Once the server is running, you can access:

Interactive API Documentation (Swagger UI): http://127.0.0.1:8000/docs
Alternative API Documentation (ReDoc): http://127.0.0.1:8000/redoc
